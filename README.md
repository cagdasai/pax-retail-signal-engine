# PAX Retail Signal Engine V2

Bu sürüm sektörlü 296 firma listesi ve 19 kaynak site ile çalışır.

## Dosyalar
- `index.html`: Yönetim paneli
- `takip_listesi.json`: Sektörlü takip listesi
- `kaynak_listesi.json`: Taranacak siteler
- `tara.py`: GitHub Issue oluşturan script
- `gorulen_haberler.json`: Duplicate önleme
- `.github/workflows/pax-retail-signal-engine-v2.yml`: GitHub Actions

## Kurulum
Bu dosyaları mevcut repo kök dizinine yükle. Sonra Actions > PAX Retail Signal Engine V2 > Run workflow.
